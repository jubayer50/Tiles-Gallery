## Tiles gallery

Tiles gallery is an amazing project for me; I implemented a lot of features and functions in this project. The most important and nice thing is the authentication, which I implemented in this project. On the home page, you can see a nice banner with a browser now button as the background of the banner. I use some images, which are changing one by one. I used here react swiper Library that looks very cool. Clicking on the browse button, you can go to the all tiles page, where you can see all our tiles. On the home page, you can also see 4 tiles card thats our featured tiles.

---

On the all-tiles page, you can see all tile products with a nice view details button beside that, there is some information about tiles like category, material, price, etc. To get more information about those specific tiles. If you click on the view detail button, it will give more information about tiles.

---

The most interesting feature on this project is the private page (authorization), my profile page, where you can see your personal information. Also, you can update your profile. Another private page is the tile's details page, where you can see all of the information about the tile

---

At the beginning, I mentioned Authentication for this. I use Better Auth, it is the most popular and Secure for authentication. I use a two-login system for this project login with Google and email password.

### There are some features of this project:

1. Login system
2. Search option on all tiles page
3. Profile update section
4. Without a login, you cannot access the tile's details page and profile page.

### The technology is used in this project

1. Next.js
2. Tailwind CSS
3. Hero UI for component library
4. React Toastify
5. React swiper etc.
